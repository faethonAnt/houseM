# Property Management System 

Full-stack web application για τη διαχείριση ακινήτων και χρηστών, υλοποιημένη με **Java Spring**.

> Ομαδικό project στα πλαίσια πανεπιστημιακού μαθήματος (3 φοιτητές).  

---

✔ User Authentication (login / roles)  
✔ CRUD Operations για ακίνητα  
✔ CRUD Operations για χρήστες  
✔ Αποθήκευση δεδομένων σε βάση δεδομένων  
✔ Βασικό Dashboard για overview

---
